# 🔐 Web Vulnerability Scanner (XSS)
### Elevate Labs Cyber Security Internship - Task 3

---

## 📌 Project Overview

This project is a **GUI-based Web Vulnerability Scanner** developed in Python to detect **Reflected Cross-Site Scripting (XSS)** vulnerabilities in web applications.

The tool allows users to input a target URL and parameter, inject common XSS payloads, and analyze responses to identify potential vulnerabilities.

---

## 🎯 Objective

- Detect basic **XSS vulnerabilities**
- Test web inputs using **crafted payloads**
- Provide **clear and accurate results**
- Generate a **professional PDF report**

---

## ⚙️ Features

✔ GUI-based interface (Tkinter)  
✔ Real-time scanning output  
✔ Multiple XSS payload testing  
✔ Accurate vulnerability detection  
✔ Error handling for stable execution  
✔ Save results as PDF report  
✔ Clean and easy-to-use design  

---

## 🛠️ Technologies Used

- Python 3
- Tkinter (GUI)
- Requests (HTTP requests)
- ReportLab (PDF generation)
