# 🔍 GUI Port Scanner (Python)

## 📌 Project Overview

This project is a **GUI-based Port Scanner** built using Python.
It scans a target IP address or domain to identify **open ports** and helps understand network exposure and security risks.

The tool provides a **modern graphical interface**, fast scanning using threads, and the ability to **export results as a PDF report**.

---

## 🎯 Objective

The objective of this project is to:

* Learn how port scanning works
* Understand open and closed ports
* Build a basic cybersecurity tool using Python
* Implement a professional GUI application

---

## 🚀 Features

* 🔹 Scan any IP address or domain
* 🔹 Custom port range selection
* 🔹 Fast scanning using multithreading
* 🔹 Displays open ports in real-time
* 🔹 Save scan results as a **PDF report**
* 🔹 Clean and professional GUI design
* 🔹 Error handling for invalid inputs

---

## 🛠️ Technologies Used

* Python
* Tkinter (GUI)
* Socket Programming
* Threading
* ReportLab (PDF generation)

---

## ▶️ How to Run

### 1. Install Requirements

```bash
pip install reportlab
```

### 2. Run the Program

```bash
python scanner.py
```

---

## 🖥️ How It Works

1. Enter the target IP or domain
2. Enter the port range (e.g., 1–1000)
3. Click **Start Scan**
4. View open ports in the result window
5. Click **Save as PDF** to export results
