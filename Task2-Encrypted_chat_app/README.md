# 🔐 Encrypted Chat Application (Python)

## 📌 Project Overview

This project is a **secure encrypted chat application** developed using Python.
It allows multiple clients to communicate with each other in real-time using a **client-server architecture**, where all messages are **end-to-end encrypted**.

The application also features a **modern GUI interface**, making it user-friendly and visually appealing.

---

## 🎯 Objective

* To understand secure communication using encryption
* To implement client-server networking using sockets
* To build a real-time encrypted messaging system
* To enhance cybersecurity concepts through practical implementation

---

## 🚀 Features

* 🔹 Real-time messaging between multiple clients
* 🔹 Client-server architecture
* 🔹 End-to-end encryption using **AES (Fernet)**
* 🔹 Secure key-based communication
* 🔹 Modern GUI (Tkinter-based)
* 🔹 Multi-client support
* 🔹 Incoming & outgoing message display

---

## 🛠️ Technologies Used

* Python
* Socket Programming
* Threading
* Tkinter (GUI)
* Cryptography (Fernet AES Encryption)

## ▶️ How to Run

### 1. Install Requirements

```bash
pip install -r requirements.txt
```

### 2. Start Server

```bash
python server.py
```

### 3. Start Clients (in multiple terminals)

```bash
python client.py
```

---

## 🖥️ How It Works

1. Start the server
2. Launch multiple clients
3. Clients connect to the server
4. Messages are encrypted before sending
5. Server broadcasts messages to all clients
6. Clients decrypt and display messages
